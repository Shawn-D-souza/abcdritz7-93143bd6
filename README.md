# Ritz7 — No-Code & AI Community for Students

The official website for **Ritz7** — a student community that makes AI and no-code accessible, helping you build real projects without writing complex code.

🌐 [ritz7.ai](https://ritz7.ai)

## Tech Stack

This project is built with:

- **Vite** — Lightning-fast build tooling
- **React 18** — Component-based UI
- **TypeScript** — Type-safe development
- **Tailwind CSS** — Utility-first styling
- **Framer Motion** — Smooth animations
- **Three.js / React Three Fiber** — 3D background effects
- **shadcn/ui** — Accessible UI components

## Getting Started

### Prerequisites

- Node.js (v18+) and npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Run Locally

```sh
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev
```

The site will be available at `http://localhost:8080`.

### Build for Production

```sh
npm run build
```

## Project Structure

```
src/
├── components/     # Reusable UI components (Navbar, Hero, etc.)
├── pages/          # Route-level pages (Index, NotFound)
├── hooks/          # Custom React hooks
├── lib/            # Utility functions
└── index.css       # Global styles and design tokens
public/
└── assets/         # Brand logos and static assets
```
